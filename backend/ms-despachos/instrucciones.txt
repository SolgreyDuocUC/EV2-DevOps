INSTRUCCIONES - MS-DESPACHOS

Este es el microservicio de gestión de logística y despachos.

TECNOLOGÍAS:
- Java 21
- Spring Boot 3.4.5
- Maven

DOCKER:
- El Dockerfile utiliza una construcción multi-etapa (Maven + JRE).
- Se recomienda ejecutarlo a través del docker-compose global en la raíz del proyecto.
- Si se construye manualmente, el contexto debe ser la carpeta 'backend'.

API:
- Swagger UI disponible en: http://localhost:8081/swagger-ui/index.html
